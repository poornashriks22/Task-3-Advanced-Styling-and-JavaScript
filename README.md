# Task-3-Advanced-Styling-and-JavaScript
Responsive web project with Image Carousel, Quiz, and API integration using HTML, CSS, and JavaScript.
This project was developed as part of my Web Development Internship at Apex Planet. The goal of this task was to implement responsive web design and create interactive web components using JavaScript.

Project Overview

This web application demonstrates responsive design techniques and dynamic user interaction. The project includes an image carousel, a simple quiz module, and integration with a public API to fetch live data.

Features

• Responsive web design using CSS Media Queries  
• Interactive Image Carousel for navigating images  
• Simple Quiz module implemented using JavaScript  
• API Integration to fetch and display real-time data  
• Clean and user-friendly interface  

Technologies Used

HTML  
CSS  
JavaScript  

Learning Outcomes

Through this project, I gained practical experience in responsive layout design, JavaScript DOM manipulation, and working with public APIs to build dynamic web applications.
